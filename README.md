📊 Vrinda-Store-Sales-Analysis

📌 Overview

This project focuses on analyzing sales and order data to extract meaningful insights. The goal is to understand customer behavior, sales performance, and key business drivers using data visualization and analysis techniques.

❓ Key Business Questions

The analysis aims to answer the following questions:

Compare the sales and orders using a single chart
Identify which month had the highest sales and orders
Determine who purchased more — men or women in 2025
Analyze different order statuses in 2025
List the top 10 states contributing to sales
Understand the relationship between age and gender based on number of orders
Identify which sales channel contributes the most
Find the highest selling category
📈 Objectives
Perform data cleaning and preprocessing
Create meaningful visualizations
Derive actionable insights from the data
Help in business decision-making
🛠️ Tools & Technologies
Microsoft Excel / Google Sheets
Data Visualization (Charts, Pivot Tables)
(Optional) SQL / Python for advanced analysis
📊 Expected Outcomes
Clear comparison of sales vs orders
Identification of peak sales periods
Customer segmentation by gender and age
Regional and channel performance insights
Category-wise sales contribution
🚀 How to Use
Load the dataset into Excel / your preferred tool
Clean and preprocess the data
Create pivot tables and charts
Answer the key business questions listed above
📌 Conclusion

This project helps in understanding sales patterns and customer behavior, enabling better strategic decisions for growth.
